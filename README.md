# CSS
# Flexbox
    1. container(box)에 적용하는 속성값들이 존재
        - display
        - flex-direction
        - flex-wrap
        - flex-flow : flex-direction과 flex-wrap를 합친거
      # item배치 관련
        - justify-content(중심축에서 배치)
        - align-item(반대축에서 배치)
        - align-content(반대축에서 배치)
    2. item(container안에 들어가는)에 지정하는 속성값들이 존재
        - flex-grow 
        - flex-shrink
        - flex-basis
        - align-self
    3. 중심축과 반대축이 존재
